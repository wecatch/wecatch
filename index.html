App turbo

app-turbo 是一个用以加速建立普通 web 站点和 rest api 的后端决方案，包含:

  • 简易的 orm 封装(基于mongodb)
  • Restful Api 服务
  • 日志服务
  • session
  • 缓存(基于redis，memcached等内存数据库)
  • 异步任务队列
  • ...
